# Release notes for fbc_curation 0.1.0

![runfrog](https://raw.githubusercontent.com/matthiaskoenig/fbc_curation/develop/docs/images/icon/frog_icon_mirror-100x80-300dpi.png)

We are pleased to release the next version `runfrog` ([https://runfrog.de]()).
The project implements the FROG analysis for reproducibility of constraint-based models in Python.

For more information see [https://github.com/matthiaskoenig/runfrog](https://github.com/matthiaskoenig/runfrog).

This release includes the following changes:
- update GUI to nicegui
- update redis to 8.2
- better separation of frontend and backend
